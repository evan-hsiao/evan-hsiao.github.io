引入背景圖 #

當引入背景圖後，畫面會呈現三個區域，中間區域為選定的背景，其中會出現幾個軸向顏色。

  • 左側軸向選項 :
    1. Number of vanishing points 軸向數量 ( 通常只能 1 或 2 個 )。
    2. Vanishing point axes 軸向:
      • 這裡會有 x、y、z 軸及正負向，共6個軸向。
    3. Reference distance
    4. Principal point
    5. 3D guide 設定3D原始點視覺效果 (關閉/盒子狀態/平面狀態)( 引入3D軟體是看不見的 )。
    6. Dim image 勾選時，中間區域顯示反黑 ( 比較能讓軸線更顯明 )。

  • 右側則是軸向數據 原則上這邊區域幾乎快用不上，除非你在圖像上要求角度要很精準，才會使用到。

  • 調整軸向時，可配合Shift鍵持續按壓，會跟著滑鼠拖曳出現放大檢視，方便對應。

最終成品，這是最後導入Blender內合成後的效果，可以看到角度上與實際背景有相當大的貼合，其餘就差別在燈光與材質上的調整，可以更貼近真實性。
